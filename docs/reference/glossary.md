# 용어 사전

## A~Z

| 용어 | 쉬운 설명 |
|------|----------|
| **CLS** (Cumulative Layout Shift) | 페이지 로딩 중 화면이 갑자기 밀리는 현상. 낮을수록 좋음. |
| **Core Web Vitals** | 구글이 정한 3가지 핵심 성능 지표 (LCP, FID, CLS). |
| **CPC** (Cost Per Click) | 광고 1회 클릭당 단가. 높을수록 수익 좋음. |
| **CTR** (Click Through Rate) | 노출 대비 클릭 비율. 검색 결과에서 얼마나 클릭하는지. |
| **DNS** | 도메인 이름을 서버 주소로 변환하는 시스템. 전화번호부 같은 것. |
| **FCP** (First Contentful Paint) | 페이지에 첫 콘텐츠가 보이는 시간. |
| **FID** (First Input Delay) | 사용자가 클릭했을 때 반응하는 속도. |
| **Git** | 코드 버전 관리 도구. 파일의 변경 이력을 추적. |
| **GitHub** | Git 기반 코드 저장소. 내 건물의 설계도를 보관하는 금고. |
| **HTTPS** | 보안 연결. 주소창에 🔒 자물쇠가 뜨는 것. |
| **LCP** (Largest Contentful Paint) | 페이지의 가장 큰 요소가 뜨는 시간. 2.5초 이하가 합격. |
| **Lighthouse** | 구글이 만든 웹사이트 품질 측정 도구. 400점 만점. |
| **LSI Keywords** | 메인 키워드와 관련된 보조 키워드. 자연스러운 글에 포함됨. |
| **MAU** (Monthly Active Users) | 월간 순방문자 수. |
| **Next.js** | React 기반 웹 프레임워크. 우리 블로그의 엔진. |
| **Open Graph** | SNS에 공유될 때 보이는 제목/설명/이미지를 지정하는 태그. |
| **RPM** (Revenue Per Mille) | 1,000회 노출당 수익. |
| **Schema.org** | 구글에 추가 정보를 알려주는 구조화 데이터. 리치 스니펫에 필요. |
| **SEO** (Search Engine Optimization) | 검색엔진 최적화. 구글에서 상위 노출되게 만드는 모든 작업. |
| **Serverless** | 서버를 직접 관리하지 않는 구조. Vercel이 대신 해줌. |
| **SSG** (Static Site Generation) | 미리 HTML을 만들어놓는 방식. 빠르고 안전. |
| **Vercel** | Next.js 사이트를 자동 배포해주는 서비스. 무료. |
| **WebP** | 구글이 만든 이미지 포맷. JPG/PNG보다 30~50% 작음. |

## ㄱ~ㅎ

| 용어 | 쉬운 설명 |
|------|----------|
| **구조화 데이터** | 구글에 "이건 블로그 글이야, 작성자는 누구야" 등을 알려주는 코드. |
| **내부 링크** | 내 사이트 안에서 다른 글로 연결하는 링크. 개미지옥 구조. |
| **다크모드** | 화면을 어두운 색상으로 바꾸는 기능. |
| **도메인** | 사이트 주소. (예: google.com) |
| **리치 스니펫** | 구글 검색 결과에 별점, 날짜 등 추가 정보가 보이는 것. |
| **메타 태그** | HTML에 숨겨진 사이트 정보. 검색 결과 제목/설명에 사용됨. |
| **반응형 디자인** | PC, 태블릿, 모바일 모두에서 깨지지 않는 디자인. |
| **사이트맵** | 내 사이트의 모든 페이지 목록. 구글에 제출하는 지도. |
| **색인 (인덱싱)** | 구글이 내 페이지를 검색 결과에 등록하는 것. |
| **시맨틱 태그** | HTML에서 의미를 가진 태그 (header, main, article 등). |
| **전환율** | 방문자 중 실제로 구매/가입한 비율. |
| **체류 시간** | 방문자가 내 사이트에 머무른 시간. 길수록 SEO에 유리. |
| **커밋** | 코드 변경사항을 GitHub에 저장하는 행위. |
| **테크니컬 SEO** | 사이트의 기술적 품질을 최적화하는 것 (속도, 구조, 접근성). |
| **콘텐츠 SEO** | 키워드 선정 + 양질의 글쓰기로 검색 상위 노출하는 것. |
| **황금 키워드** | 검색량은 높고 경쟁은 낮은 키워드. 트래픽 독점 가능. |
