# 트러블슈팅

> 에러가 나면 여기를 먼저 보세요. 당신이 겪을 시행착오, 제가 미리 다 겪었습니다.

---

## 🔴 설치 & 환경 문제

### `npx` 명령어가 안 먹혀요

**원인:** Node.js 미설치

**해결:**
1. [nodejs.org](https://nodejs.org) 접속
2. **LTS 버전** 다운로드 & 설치
3. 터미널 재시작
4. `node --version` 으로 확인

---

### `npm run dev`가 안 돼요

**원인:** 의존성 미설치

```bash
cd my-blog
npm install    # 의존성 설치
npm run dev    # 다시 실행
```

---

### localhost:3000에 아무것도 안 떠요

**체크리스트:**
1. 터미널에 에러 메시지가 있는지 확인
2. 포트 3000이 이미 사용 중인지 확인
3. 브라우저 캐시 삭제 후 재접속

```bash
# 포트 3000 사용 중인 프로세스 확인 (Mac)
lsof -i :3000

# 다른 포트로 실행
npm run dev -- -p 3001
```

## 🟡 배포 문제

### Vercel 배포가 실패해요

**흔한 원인:**

| 에러 | 해결 |
|------|------|
| Build failed | Vercel 로그에서 에러 확인 → Claude Code에 붙여넣기 |
| Module not found | `npm install` 후 다시 커밋 |
| Type error | Claude Code에게 "이 타입 에러 고쳐줘" |

---

### CSS가 배포 후에 안 먹혀요

**원인:** 경로 문제

```
❌ <link href="/css/style.css">      (절대 경로)
✅ <link href="./css/style.css">     (상대 경로)
```

---

### 커스텀 도메인이 연결 안 돼요

1. DNS 설정 후 **최대 48시간** 기다리기
2. Vercel 대시보드에서 도메인 상태 확인
3. DNS 레코드가 정확한지 재확인

---

## 🟢 SEO 문제

### 구글에 내 사이트가 안 떠요

| 원인 | 해결 |
|------|------|
| 아직 인덱싱 안 됨 | 서치 콘솔 → URL 검사 → 색인 요청 |
| sitemap 미제출 | 서치 콘솔 → Sitemaps → 제출 |
| robots.txt 차단 | `Disallow: /` 가 있는지 확인 |
| 사이트가 너무 새로움 | 1~2주 기다리기 (정상) |

---

### Lighthouse 점수가 안 올라요

| 감점 요인 | 확인 방법 | 해결 |
|----------|----------|------|
| 이미지 최적화 | Performance 탭 | WebP 변환 + lazy loading |
| CLS | Performance 탭 | 이미지에 width/height 추가 |
| 접근성 | Accessibility 탭 | alt 태그, 색상 대비 확인 |
| 외부 폰트 | Network 탭 | 시스템 폰트로 교체 |

---

## 🔵 수익화 문제

### 애드센스 거절됐어요

| 사유 | 해결 |
|------|------|
| "유용하지 않은 콘텐츠" | 글 15개 이상 + 각 2,000자 이상 |
| "콘텐츠 부족" | 개인정보처리방침/연락처 페이지 추가 |
| "사이트 디자인" | 네비게이션 정리, 카테고리 추가 |

> 거절은 흔합니다. 수정 후 재신청하면 대부분 통과.

---

### 수익이 너무 적어요

| 상황 | 원인 | 해결 |
|------|------|------|
| 방문자 100명 미만 | 글이 적음 | 30개 이상 목표 |
| 방문자 많은데 수익 적음 | CPC 낮은 키워드 | 수익성 높은 키워드로 전환 |
| 광고 클릭 안 됨 | 광고 위치 비효율 | 자동 → 수동 광고 전환 |

## 🧠 만능 해결법

어떤 에러든 이 프로세스를 따르세요:

```
1. 에러 메시지를 그대로 복사
2. Claude Code에 붙여넣기
3. "이 에러를 해결해줘" 입력
4. 수정된 코드로 교체
5. 재실행
```

> AI 코딩의 정상적인 과정입니다.\
