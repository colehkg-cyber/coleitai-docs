# 프롬프트 모음집

> 강의에서 사용한 모든 프롬프트를 한곳에 모았습니다. 복사해서 바로 쓰세요.

---

## 🏗️ 사이트 구축용

### 디자인 커스텀

```
메인 컬러를 [색상코드]로 바꿔줘.
헤더, 링크, 버튼 색상 전부 통일.
Lighthouse 점수에 영향 없도록.
```

```
다크모드 토글 버튼을 헤더 우측에 추가해줘.
- 아이콘: 🌙 / ☀️
- 사용자 설정 localStorage 저장
- 부드러운 전환 애니메이션
```

```
블로그 글 목록을 카드형 레이아웃으로 바꿔줘.
- 데스크탑 3열 / 태블릿 2열 / 모바일 1열
- 카드에 썸네일, 제목, 날짜, 요약 표시
```

### SEO 설정

```
구글 서치 콘솔 인증 파일 [파일명].html을
public 폴더에 추가해줘.
```

```
구글 애드센스 코드를 내 Next.js 블로그에 삽입해줘.
ads.txt 파일도 public 폴더에 만들어줘.
```

## 📝 글쓰기용

### 만능 글쓰기 프롬프트

```
"[키워드]"로 블로그 글을 써줘.

[구조]
- H1: 글 제목 (60자 이내, 키워드 포함)
- 도입부: 독자의 문제점 + "이 글에서 해결됩니다" (200자)
- H2 섹션 4~6개 (각 500자 내외)
- 각 섹션에 표/리스트/코드블록 중 하나 이상
- FAQ 3~5개
- 결론: 핵심 3줄 + CTA

[SEO]
- 키워드: H1, 도입부, H2 1개, 결론에 자연스럽게 포함
- 메타 디스크립션 1줄 (155자 이내) 별도 제공
- 내부 링크 위치 2곳 표시

[톤앤매너]
- 전문적이지만 친근한 해요체
- 어려운 용어는 괄호 안에 쉬운 설명
- 구체적 숫자 포함
- 2,500~4,000자
```

### 사람 냄새 추가

```
위 글에 다음을 자연스럽게 추가해줘:
1. 개인적인 사용 경험 1~2개
2. "솔직히", "개인적으로는" 같은 구어체
3. 독자에게 말 거는 문장
4. 반전이나 의외의 포인트 1개
```

### 제휴 마케팅 글

```
"[제품명]" 리뷰 블로그 글을 써줘.
- 솔직한 장단점 (신뢰감)
- "추천 / 비추" 대상 구분
- 경쟁 제품 비교표
- 자연스러운 구매 유도 (강매 금지)
```

## 🔍 키워드 분석용

### 경쟁 분석

```
"[키워드]"를 구글에 검색했을 때
상위 5개 글의 장단점을 분석해줘.

각 글에 대해:
- 작성 시점
- 글 길이
- 다루는 내용
- 빠져 있는 정보
- 내가 이길 수 있는 포인트
```

### 검색 의도 분석

```
"[키워드]"를 검색하는 사람의 의도를 분석해줘.

1. 주요 검색 의도 (정보형/비교형/구매형)
2. 검색자가 진짜 알고 싶은 것 3가지
3. 최적의 글 구조
4. 반드시 포함할 정보
5. 경쟁 글에서 빠져있는 정보
```

2. 추가할 최신 정보 제안

4. 수정된 전체 글 작성
