# Step 5. 황금 키워드 채굴

> 사람들이 검색은 하는데, 읽을 만한 문서는 없는 '빈 땅'을 찾아냅니다.

---

## 황금 키워드 공식 (복습)

```
검색량 HIGH  ×  경쟁도 LOW  =  황금 키워드 🏆
```

---

## 1단계: 데이터 마이닝

### 도구별 사용법

#### 구글 트렌드 (무료)

> **용도:** 키워드의 검색 추세 확인

1. [trends.google.co.kr](https://trends.google.co.kr) 접속
2. 키워드 입력 (예: "맥북 에어 M3")
3. 확인할 것:
   - 검색량이 **상승 추세**인가? ✅
   - 계절성이 있는가? (특정 시기에만 검색되는 키워드 주의)
   - 관련 주제/검색어에 추가 키워드 힌트

#### 구글 키워드 플래너 (무료)

> **용도:** 월간 검색량 수치 확인

1. [ads.google.com](https://ads.google.com) → 도구 → 키워드 플래너
2. "새 키워드 찾기" → 키워드 입력
3. 필터: **월간 검색량 1,000 이상**

#### AnswerThePublic (무료 일부)

> **용도:** 사람들이 실제로 어떤 질문을 하는지 확인

1. [answerthepublic.com](https://answerthepublic.com) 접속
2. 키워드 입력 → 한국어 선택
3. "왜", "어떻게", "vs" 등 질문형 키워드 수집

---

## 2단계: 문서 공백 발견

> 경쟁 분석 = 구글에서 직접 검색해보기

### 체크 프로세스

```
키워드를 구글에 검색
    ↓
1페이지 결과 분석
    ↓
아래 조건 중 2개 이상 해당하면 → 황금 키워드! 🏆
```

| # | 체크 항목 | 해당하면 기회 |
|---|----------|------------|
| 1 | 상위 글이 **3년 이상** 오래됨 | ✅ 최신 글로 대체 가능 |
| 2 | 상위 글 내용이 **부실하거나 짧음** | ✅ 더 좋은 글로 이기기 가능 |
| 3 | 상위 결과가 **대형 사이트가 아님** | ✅ 개인 블로그도 진입 가능 |
| 4 | **"관련 검색어"에 답변이 없음** | ✅ FAQ 포함하면 추가 노출 |
| 5 | 상위 글의 **Lighthouse 점수가 낮음** | ✅ 400점이면 구조적 우위 |

### AI로 경쟁 분석하기

```
"맥북 에어 M3 발열 해결법"을 구글에 검색했을 때
상위 5개 글의 장단점을 분석해줘.

각 글에 대해:
- 작성 시점
- 글 길이
- 다루는 내용
- 빠져 있는 정보
- 내가 이길 수 있는 포인트
```

---

## 3단계: 키워드 리스트 관리

### 간단한 스프레드시트

| 키워드 | 월 검색량 | 경쟁도 | 기회도 | 글 작성 |
|--------|----------|--------|--------|--------|
| 맥북 에어 M3 발열 해결법 | 2,400 | 낮음 | ★★★ | ⬜ |
| 아이폰 16 배터리 절약 | 5,100 | 중간 | ★★☆ | ⬜ |
| Claude Code 사용법 | 3,200 | 낮음 | ★★★ | ⬜ |

> 💡 **최소 20~30개** 키워드를 확보한 후, 기회도가 높은 것부터 글을 씁니다.

---

## 실전 예시: 황금 키워드 발굴

```
키워드: "맥북 에어 M3 발열 해결법"
─────────────────────────────────
구글 트렌드:    상승 추세 ✅
월 검색량:      2,400회 ✅
경쟁 분석:
  - 1위 글: 2023년 작성 (2년 전) → 오래됨 ✅
  - 2위 글: 500자 짧은 글 → 부실 ✅  
  - 3위 글: 네이버 블로그 → Lighthouse 낮음 ✅
─────────────────────────────────
판정: 🏆 황금 키워드!
→ 최신 정보 + 2000자 이상 + 400점 사이트
→ 1페이지 진입 가능성 매우 높음
```

## ✅ 체크리스트

- [ ] 구글 트렌드에서 내 분야 키워드 트렌드 확인
- [ ] 키워드 플래너로 월간 검색량 확인
- [ ] 경쟁 분석 완료 (상위 5개 글 체크)
- [ ] 황금 키워드 최소 10개 확보
- [ ] 키워드 리스트 스프레드시트 정리
