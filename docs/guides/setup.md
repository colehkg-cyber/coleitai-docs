# Step 1. 환경 세팅

> 도구 3가지만 준비하면 됩니다. 10분이면 끝납니다.

---

## 뭘 하는 건가요?

블로그(건물)를 짓기 위한 **도구 3개**를 준비하는 단계입니다.

비유하면:
- **Claude** = 건축가. "이런 건물 지어줘" 하면 설계도(코드)를 짜줍니다.
- **Github** = 금고. 설계도를 안전하게 보관합니다.
- **Vercel** = 건물 관리인. 설계도를 받아서 실제 건물(사이트)을 인터넷에 올려줍니다.

이 3개를 연결하면 마법이 일어납니다:\
**글을 쓰고 저장하면 → 자동으로 사이트가 업데이트됩니다.**

---

## 필요한 것

| # | 도구 | 뭘 해주나요? | 비용 | 소요 시간 |
|---|------|------------|------|----------|
| 1 | **Claude** | AI가 코드를 짜줌 (건축가) | Pro $20/월 | 3분 |
| 2 | **Github** | 코드를 저장해줌 (금고) | 무료 | 3분 |
| 3 | **Vercel** | 사이트를 인터넷에 올려줌 (관리인) | 무료 | 3분 |

> 💰 세 개 합쳐서 **월 $20 (약 2만 7천 원).** 이게 전부입니다.\
> 나머지는 전부 무료!

## 1. Claude 가입

Claude는 **AI 도우미**입니다.\
여러분이 "이거 만들어줘"라고 말하면, AI가 코드를 짜줍니다.

### 가입 방법 (3분)

1. 인터넷 브라우저에서 [claude.ai](https://claude.ai)에 접속합니다
2. **"Sign Up"** (가입) 버튼을 클릭합니다
3. 이메일 주소를 입력하고 비밀번호를 만듭니다
4. 이메일로 온 인증 링크를 클릭합니다
5. 로그인하면 대화창이 나옵니다 — 이게 AI 건축가예요!

### Pro 플랜 구독하기

6. 왼쪽 아래 **"Upgrade to Pro"**를 클릭합니다
7. 결제 정보를 입력하고 **Pro 플랜** (월 $20)을 구독합니다

### Claude가 하는 일 (예시)

```
여러분이 하는 것:    "헤더에 다크모드 버튼 만들어줘"
Claude가 하는 것:    완성된 코드를 만들어줌 ✅
```

코딩을 몰라도 됩니다. **한국어로 원하는 걸 말하면** AI가 코드로 바꿔줍니다.

:::warning 💡 무료 플랜도 되나요?
무료 플랜으로도 시작할 수 있지만, 코드 생성 횟수에 제한이 있습니다.\
원활한 수강을 위해 **Pro 플랜을 추천**합니다.\
나중에 블로그가 잘 되면 이 비용은 광고 수익으로 충분히 커버됩니다!
:::

## 2. Github 가입

Github(깃허브)은 **코드를 저장하는 곳**입니다.\
쉽게 말하면, 건물 설계도를 넣어두는 **금고**예요.

### 왜 필요한가요?
- 코드를 안전하게 보관합니다 (컴퓨터가 고장나도 괜찮아요)
- 잘못 수정했으면 **이전 버전으로 되돌리기** 가능
- Vercel과 연동하면 코드를 올릴 때마다 **사이트가 자동 업데이트**

### 가입 방법 (3분)

1. [github.com](https://github.com)에 접속합니다
2. **"Sign up"** 버튼을 클릭합니다
3. 이메일 주소를 입력합니다
4. 비밀번호를 만듭니다 (영문 + 숫자 조합)
5. 사용자 이름(username)을 정합니다
   - 이게 나중에 사이트 주소에 들어가요 (예: github.com/**내아이디**)
   - 짧고 기억하기 쉬운 걸로 추천!
6. 이메일 인증을 완료합니다
7. **무료 플랜**을 선택합니다 (유료 안 해도 됩니다!)

> ✅ 끝! 이게 여러분의 코드 금고입니다.

---

## 3. Vercel 가입

Vercel(버셀)은 **사이트를 인터넷에 올려주는 서비스**입니다.\
쉽게 말하면, 건물 관리인이에요.

Github에 코드를 올리면 → Vercel이 자동으로 감지해서 → 사이트를 전 세계에 공개합니다.

### 가입 방법 (3분)

1. [vercel.com](https://vercel.com)에 접속합니다
2. **"Sign Up"** 버튼을 클릭합니다
3. ⭐ **"Continue with GitHub"**를 클릭합니다
   - 이게 중요합니다! Github 계정으로 연동해야 자동 배포가 됩니다
4. Github 로그인 화면이 나오면 로그인합니다
5. **"Hobby"** 플랜을 선택합니다 (무료!)

### Vercel이 해주는 것

```
┌──────────┐      자동!      ┌──────────┐     결과
│  GitHub  │ ──→  배포  ──→  │  Vercel  │ ──→  내 사이트가
│ 코드 저장  │                │ 자동 배포  │     전 세계에 공개!
└──────────┘                 └──────────┘
```

- 📦 개인 프로젝트 **평생 무료**
- 🌐 트래픽 **무제한**
- 🔒 HTTPS 보안 인증서 **무료** (사이트 주소 앞에 🔒 자물쇠가 붙어요)
- 🚀 전 세계 CDN 자동 적용 (어느 나라에서 접속해도 빠름)

> 💡 예전에는 이런 걸 하려면 서버를 빌려야 했고, 월 5~10만 원이 들었습니다.\
> 지금은 **무료**입니다. 세상 좋아졌어요!

## 4. (선택) 도메인 구매

도메인이란, **"myblog.com"** 같은 나만의 주소입니다.

### 지금 당장 안 해도 됩니다!

도메인 없이도 시작 가능합니다.\
Vercel이 **무료 주소**를 줍니다: `my-blog.vercel.app`

나중에 도메인을 사서 연결해도 됩니다. 언제든 가능해요.

### 하고 싶다면?

| 서비스 | 연간 비용 | 특징 |
|--------|----------|------|
| [Namecheap](https://namecheap.com) | ~₩15,000 | 글로벌, 가성비 좋음 |
| [가비아](https://gabia.com) | ~₩18,000 | 한국어 지원, 한국 서비스 |
| [Cloudflare](https://cloudflare.com) | ~₩12,000 | 가장 저렴, 영문 서비스 |

> 💡 도메인은 **연간** 비용입니다. 한 달이 아니에요!\
> 1년에 1만 5천 원이면 하루에 40원 정도입니다.

## 전체 구조 — 한눈에 보기

```
여러분 (건물주)
    │
    │ "이거 만들어줘"
    ↓
┌──────────┐
│  Claude  │ ← AI 건축가 (코드를 짜줌)
│  월 $20  │
└────┬─────┘
     │ 코드를 저장
     ↓
┌──────────┐
│  Github  │ ← 금고 (코드를 안전하게 보관)
│   무료   │
└────┬─────┘
     │ 자동 감지
     ↓
┌──────────┐
│  Vercel  │ ← 관리인 (사이트를 인터넷에 공개)
│   무료   │
└────┬─────┘
     │
     ↓
  🌐 내 블로그가 전 세계에 공개!
```

**한 번 세팅하면 끝!**\
이후에는 글 쓰고 저장만 하면 자동으로 사이트가 업데이트됩니다.
