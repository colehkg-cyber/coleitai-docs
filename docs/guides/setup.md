# Step 1. 환경 세팅

> 도구 3가지만 준비하면 됩니다. 10분이면 끝납니다.

---

## 필요한 것

| # | 도구 | 용도 | 비용 | 소요 시간 |
|---|------|------|------|----------|
| 1 | **Claude** | AI가 코드를 짜줌 | Pro $20/월 | 3분 |
| 2 | **Github** | 코드 저장소 (내 건물의 설계도) | 무료 | 3분 |
| 3 | **Vercel** | 배포 + 호스팅 (건물 관리인) | 무료 | 3분 |

> 💡 세 개 합쳐서 월 $20. 이게 전부입니다.

---

## 1. Claude 가입

### 1-1. 계정 생성

1. [claude.ai](https://claude.ai) 접속
2. 이메일로 회원가입
3. **Pro 플랜** 구독 (월 $20)

### 1-2. Claude Code 활성화

Claude Code는 **자연어를 코드로 바꿔주는 도구**입니다.

```
당신이 하는 것:    "헤더에 다크모드 버튼 만들어줘"
Claude가 하는 것:  완성된 코드를 생성
```

:::warning
**무료 플랜으로도 시작할 수 있지만**, 코드 생성 횟수에 제한이 있습니다.\
원활한 수강을 위해 Pro 플랜을 권장합니다.
:::

---

## 2. Github 가입

### 2-1. 계정 생성

1. [github.com](https://github.com) 접속
2. **Sign up** 클릭
3. 이메일, 비밀번호 입력
4. 무료 플랜 선택

### 2-2. Github란?

> 내 코드(설계도)를 안전하게 저장하는 금고.

- 코드를 올리면 → 자동으로 버전 관리
- 잘못 수정했으면 → 이전 버전으로 되돌리기 가능
- Vercel과 연동하면 → 코드 올릴 때마다 사이트 자동 업데이트

---

## 3. Vercel 가입

### 3-1. 계정 생성

1. [vercel.com](https://vercel.com) 접속
2. **Sign Up** → **Continue with GitHub** 클릭 (Github 계정으로 연동)
3. **Hobby** 플랜 선택 (무료)

### 3-2. Vercel이란?

> 내 사이트를 전 세계에 배포해주는 건물 관리인.

```
┌──────────┐      Auto       ┌──────────┐     Result
│  GitHub  │ ──Deploy──→     │  Vercel  │ ──→  내 사이트가
│ 코드 저장  │                │ 자동 배포  │     전 세계에 공개!
└──────────┘                 └──────────┘

비용: $0 (Hobby 플랜)
- 개인 프로젝트 평생 무료
- 트래픽 무제한
- HTTPS 보안 인증서 무료
- 전 세계 CDN 자동 적용
```

:::info
**기존 방식(AWS EC2)** 대비 월 5~10만원 절감.\
서버 터질 걱정 없는 Serverless 구조입니다.
:::

## 4. (선택) 도메인 구매

자체 도메인이 없어도 시작할 수 있습니다.\
Vercel이 `your-blog.vercel.app` 주소를 무료로 줍니다.

하지만 **진지하게 할 거라면** 도메인 구매를 권장합니다:

| 서비스 | 연간 비용 | 특징 |
|--------|----------|------|
| Namecheap | ~₩15,000 | 글로벌 가성비 |
| 가비아 | ~₩18,000 | 한국어 지원 |
