# Step 6. AI 글쓰기 실전

> AI가 초안을 쓰고, 내가 경험을 얹고, 구글이 띄워줍니다.

---

## AI 글쓰기 워크플로우

```
1. 황금 키워드 선정 (Step 5에서 완료)
       ↓
2. AI에게 글 구조 요청
       ↓
3. AI가 초안 작성
       ↓
4. 내가 검수 + 개인 경험 추가
       ↓
5. 블로그에 발행 → Github 커밋 → 자동 배포
       ↓
6. 구글 서치 콘솔에서 색인 요청
```

---

## 글 구조 요청 프롬프트

### 프롬프트 (그대로 복사)

```
"[키워드]"로 블로그 글을 써줘.

[구조]
- H1: 글 제목 (60자 이내, 키워드 포함)
- 도입부: 독자의 문제점 짚기 + "이 글에서 해결됩니다" 약속 (200자)
- H2 섹션 4~6개 (각 500자 내외)
- 각 섹션에 표, 리스트, 코드블록 중 하나 이상 포함
- FAQ 3~5개 (구조화 데이터용)
- 결론: 핵심 요약 3줄 + CTA

[SEO 규칙]
- 키워드를 H1, 도입부, H2 1개, 결론에 자연스럽게 포함
- 메타 디스크립션용 요약 1줄 (155자 이내) 별도 제공
- 내부 링크 삽입 위치 2곳 표시 ([내부 링크])

[톤앤매너]
- 전문적이지만 친근한 해요체
- 어려운 용어는 괄호 안에 쉬운 설명
- 구체적 숫자와 데이터 포함

[글자 수]
- 2,500~4,000자
```

## AI 글에 사람 냄새 추가

AI가 쓴 글은 **너무 깔끔해서** 구글이 의심할 수 있습니다.\
아래를 반드시 추가하세요.

### 추가할 것 5가지

| # | 추가 요소 | 예시 |
|---|----------|------|
| 1 | **개인 경험** | "제가 직접 써보니까..." |
| 2 | **실패담** | "처음에 이렇게 했다가 망했습니다" |
| 3 | **의견** | "솔직히 이건 별로라고 생각합니다" |
| 4 | **스크린샷** | 실제 화면 캡처 이미지 |
| 5 | **구어체 표현** | "진짜 이거 대박이에요" |

### 프롬프트로 보강하기

```
위 글에 다음을 자연스럽게 추가해줘:
1. 개인적인 사용 경험 1~2개
2. "솔직히", "개인적으로는" 같은 구어체
3. 독자에게 말 거는 문장 ("혹시 이런 경험 있지 않나요?")
4. 반전이나 의외의 포인트 1개
```

## 글 발행하기

### 우리 템플릿에서 글 추가하는 법

```bash
# 1. 글 파일 생성 (마크다운)
# /posts/맥북-에어-m3-발열-해결법.mdx

# 2. 상단에 메타 정보 작성
---
title: "맥북 에어 M3 발열 해결법 5가지 (2026년 최신)"
description: "맥북 에어 M3 발열 문제, 이 5가지 방법으로 즉시 해결하세요."
date: "2026-03-13"
category: "맥북"
thumbnail: "/images/macbook-m3.webp"
---

# 3. 본문 작성 (마크다운 형식)

# 4. Github 커밋 → Vercel 자동 배포
git add .
git commit -m "새 글: 맥북 에어 M3 발열 해결법"
git push
```

### 발행 후 해야 할 것

1. **구글 서치 콘솔** → URL 검사 → 색인 생성 요청
2. **24시간 이내** 구글에 인덱싱됨 (400점 사이트 특권)

## 체류 시간 늘리는 글쓰기 팁

| 기법 | 설명 | 예시 |
|------|------|------|
| **호기심 갭** | 다음 내용 궁금하게 | "하지만 진짜 중요한 건 다음입니다" |
| **패턴 인터럽트** | 예상 깨기 | "대부분 여기서 틀립니다" |
| **구체적 숫자** | 모호함 제거 | "3.2초 만에" (X: "빠르게") |
| **표/리스트** | 시각적 쉼표 | 3문장 이상 연속하지 않기 |

## 글 발행 주기

| 시기 | 권장 주기 | 이유 |
|------|----------|------|
| 초기 (1~3개월) | 주 2~3개 | 구글 신뢰도 구축 |
| 안정기 (4개월~) | 주 1개 | 유지 + 기존 글 업데이트 |
| 자동화 단계 | 월 2~3개 | 복리 효과로 글 안 써도 트래픽 유지 |
