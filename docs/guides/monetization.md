# Step 7. 수익화 세팅

> 잠자는 동안 통장에 돈이 꽂히는 시스템을 켭니다.

---

## 수익화 로드맵

```
Level 1: 애드센스 (광고 수익)      ← 먼저 이것부터
Level 2: 제휴 마케팅 (추천 수수료)   ← 트래픽 확보 후
Level 3: 자체 상품 (디지털 제품)     ← 팬덤 형성 후
```

---

## Level 1: 구글 애드센스

### 승인 조건

| 조건 | 요구사항 | 우리 사이트 |
|------|---------|-----------|
| 고유 콘텐츠 | 오리지널 글 | ✅ AI + 개인 경험 |
| 콘텐츠 양 | 15~20개 이상 | 5~6주차면 달성 |
| 필수 페이지 | 개인정보처리방침, 연락처 | 템플릿에 포함 |
| 사이트 품질 | 깔끔한 디자인 | ✅ Lighthouse 400점 |
| HTTPS | 보안 연결 | ✅ Vercel 자동 제공 |

> 💡 **400점짜리 사이트는 애드센스 승인에 가장 유리합니다.**\
> 구글 입장에서 자기네가 만든 기준을 완벽히 충족하는 사이트니까요.

### 신청 방법

1. [adsense.google.com](https://adsense.google.com) 접속
2. 사이트 URL 입력
3. 애드센스 코드를 사이트에 삽입

```
Claude Code에게:
"구글 애드센스 코드를 내 Next.js 블로그에 삽입해줘.
ads.txt 파일도 public 폴더에 만들어줘."
```

4. 심사 대기 (보통 2일~2주)

### 광고 배치 가이드

| 위치 | 수익 효과 | 주의 |
|------|----------|------|
| 글 제목 아래 | ★★★ | 너무 크면 체류 시간 하락 |
| 본문 중간 (H2 사이) | ★★★ | 자연스러운 위치에 |
| 글 하단 | ★★☆ | 완독한 사람에게 노출 |

> 처음에는 **자동 광고**를 추천합니다. 구글이 최적 위치에 자동 배치.

---

## Level 2: 제휴 마케팅

### 개념

글에서 제품 추천 → 독자가 링크로 구매 → 수수료 수익

### 주요 플랫폼

| 플랫폼 | 수수료 | 적합한 경우 |
|--------|--------|-----------|
| 쿠팡 파트너스 | 3~7% | 제품 리뷰, 추천 글 |
| 아마존 어소시에이트 | 1~10% | 해외 트래픽 |
| SaaS 직접 제휴 | 10~50% | AI 도구, 소프트웨어 리뷰 |

### 제휴 글 쓰는 프롬프트

```
"[제품명]" 리뷰 블로그 글을 써줘.

- 솔직한 장단점 (신뢰감이 핵심)
- "이런 사람에게 추천 / 이런 사람은 비추"
- 경쟁 제품 비교표
- 자연스러운 구매 유도 (강매 금지)
```

---

## Level 3: 자체 디지털 상품

트래픽이 안정되면 자체 상품으로 확장:

| 상품 | 예상 가격 | 예시 |
|------|----------|------|
| PDF 가이드 | ₩9,900~29,900 | "SEO 체크리스트 e-book" |
| 프롬프트 팩 | ₩19,900 | "AI 글쓰기 프롬프트 50선" |
| 템플릿 | ₩29,900~49,900 | "SEO 블로그 템플릿" |
| 온라인 강의 | ₩99,000+ | 이 강의가 바로 그 예시 |

---

## 수익 시뮬레이션

월 20만 트래픽 기준:

| 수익원 | 월 예상 수익 |
|--------|------------|
| 애드센스 (RPM ₩3,000) | ₩600,000 |
| 제휴 마케팅 (전환율 0.5%) | ₩300,000 |
| 디지털 상품 (전환율 0.1%) | ₩200,000 |
| **합계** | **₩1,100,000** |

```
서버비:    ₩0
순수익률:  99%
```

> 글을 안 써도 매달 100만 원 이상이 자동으로 들어오는 시스템.
