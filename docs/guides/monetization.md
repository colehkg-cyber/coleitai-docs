# Step 7. 수익화 & 확장

> 잠자는 동안 통장에 돈이 꽂히는 시스템을 켜고, 더 크게 키웁니다.

---

## 뭘 하는 건가요?

지금까지:
- ✅ 건물(사이트) 완공
- ✅ 인테리어(디자인) 완료
- ✅ 구글에 등록
- ✅ 세입자(글) 입주

이제 **월세를 받을 시스템**을 켭니다!\
그리고 건물을 **더 크게 키우는 방법**도 알려드립니다.

---

## 수익화 3단계 로드맵

한 번에 다 하려면 안 됩니다. 순서대로 하세요!

```
Level 3: 자체 상품 판매 (디지털 제품)    ← 팬덤이 생긴 후
Level 2: 제휴 마케팅 (추천 수수료)       ← 트래픽이 쌓인 후
Level 1: 구글 애드센스 (광고 수익)        ← 🔥 먼저 이것부터!
```

건물에 비유하면:
- Level 1 = 기본 월세 (임대 수입)
- Level 2 = 1층 상가 임대 (추가 수입)
- Level 3 = 건물 옥상에 광고판 (프리미엄 수입)

---

## Level 1: 구글 애드센스

### 애드센스가 뭔가요?

**구글이 내 사이트에 광고를 자동으로 넣어주고, 돈을 줍니다.**

여러분이 광고주를 찾을 필요 없어요!\
구글이 알아서 방문자에게 맞는 광고를 보여줍니다.\
방문자가 광고를 보거나 클릭하면 → **여러분 통장에 돈이 들어옵니다.**

### 얼마나 벌 수 있나요?

| 월 트래픽 | 예상 월 수익 |
|----------|------------|
| 1,000명 | 3,000~5,000원 |
| 5,000명 | 15,000~30,000원 |
| 20,000명 | 60,000~150,000원 |
| 100,000명 | 300,000~700,000원 |

> 💡 분야에 따라 차이가 큽니다.\
> 금융, IT, 보험 분야는 광고 단가가 높고,\
> 일상, 취미 분야는 단가가 낮아요.

### 애드센스 승인 조건

애드센스는 아무나 못 씁니다. **구글의 심사**를 통과해야 해요.

| 조건 | 필요한 것 | 우리 상태 |
|------|---------|----------|
| 고유 콘텐츠 | 직접 쓴(AI+경험) 오리지널 글 | ✅ 완료 |
| 콘텐츠 양 | 글 15~20개 이상 | 6강 따라왔으면 OK |
| 필수 페이지 | 개인정보처리방침, 연락처 페이지 | ✅ 템플릿에 포함 |
| 사이트 품질 | 깔끔한 디자인, 빠른 속도 | ✅ Lighthouse 400점 |
| HTTPS | 보안 연결 | ✅ Vercel 자동 제공 |

> 🔥 **400점짜리 사이트는 애드센스 승인에 가장 유리합니다!**\
> 구글이 만든 기준을 완벽히 충족하는 사이트니까요.\
> 구글 입장에서 "이 사이트에 우리 광고 넣어도 되겠다" 하는 거예요.

### 신청 방법 (10분)

1. [adsense.google.com](https://adsense.google.com)에 접속합니다
2. Google 계정으로 로그인합니다
3. 내 사이트 URL을 입력합니다
4. 애드센스 코드가 나옵니다

이 코드를 사이트에 넣어야 하는데, Claude한테 시키면 됩니다:

```
구글 애드센스 코드를 내 블로그에 삽입해줘.
ads.txt 파일도 public 폴더에 만들어줘.
코드: [여기에 애드센스 코드 붙여넣기]
```

5. Github에 커밋 → Vercel 자동 배포
6. 서치 콘솔에서 "확인" 클릭
7. **심사 대기** (보통 2일~2주, 빠르면 하루!)

### 광고 배치 — 어디에 넣나요?

처음에는 **"자동 광고"**를 추천합니다.\
구글이 알아서 가장 좋은 위치에 광고를 배치해줘요.

나중에 수동으로 조절하고 싶으면:

| 위치 | 수익 효과 | 주의 |
|------|----------|------|
| 글 제목 아래 | ★★★ | 너무 크면 방문자가 떠남 |
| 본문 중간 | ★★★ | 자연스러운 위치에 |
| 글 하단 | ★★☆ | 끝까지 읽은 사람에게 |
| 사이드바 | ★☆☆ | 컴퓨터에서만 보임 |

> ⚠️ **광고 욕심 부리지 마세요!**\
> 광고가 너무 많으면 → 방문자가 불편해서 떠남 → 구글 순위 하락 → 트래픽 감소\
> 적당히 넣는 게 오히려 **돈을 더 법니다.**

## Level 2: 제휴 마케팅

트래픽이 월 5,000명 이상 되면 시작하세요!

### 제휴 마케팅이 뭔가요?

글에서 제품을 추천하고 → 독자가 여러분의 링크로 구매하면 → **수수료**를 받는 것!

예를 들어, "맥북 발열 해결법" 글에서 쿨링 패드를 추천합니다.\
독자가 그 링크로 쿠팡에서 사면? 가격의 3~7%가 **여러분 수익!**

### 주요 플랫폼

| 플랫폼 | 수수료 | 적합한 글 |
|--------|--------|----------|
| **쿠팡 파트너스** | 3~7% | 제품 리뷰, 추천 글 |
| **아마존 어소시에이트** | 1~10% | 영어 블로그, 해외 트래픽 |
| **SaaS 직접 제휴** | 10~50% | AI 도구, 소프트웨어 리뷰 |

> 💡 **SaaS 제휴가 수수료가 가장 높습니다!**\
> AI 도구 리뷰 블로그라면 이게 대박이에요.

### 제휴 글 쓰는 프롬프트

```
"[제품명]" 리뷰 블로그 글을 써줘.

[구조]
- 실제 사용 후기 형식
- 장점 3개, 단점 2개 (솔직하게!)
- 다른 제품과 비교
- "이런 사람에게 추천" / "이런 사람은 비추" 섹션
- [제휴 링크 삽입 위치] 표시

[주의]
- 과장 금지, 솔직한 톤
- "광고" 또는 "제휴 링크 포함" 고지 문구 추가
```

> ⚠️ **솔직해야 합니다!**\
> 단점 없이 장점만 쓰면 독자가 안 믿어요.\
> "솔직히 이건 별로인데, 이 부분은 진짜 좋다" — 이런 글이 오히려 잘 팔립니다.\
> 그리고 **제휴 링크 고지는 법적 의무**입니다. 반드시 표시하세요.

## Level 3: 자체 상품 판매

트래픽이 월 10,000명 이상, 팬층이 생겼을 때 시작하세요.

| 상품 | 예상 가격 | 예시 |
|------|----------|------|
| **전자책(PDF)** | ₩9,900~29,900 | "SEO 완벽 가이드" |
| **프롬프트 팩** | ₩19,900 | "AI 글쓰기 프롬프트 50선" |
| **템플릿** | ₩29,900~49,900 | "SEO 블로그 템플릿" |
| **온라인 강의** | ₩99,000+ | 이 강의가 바로 그 예시! |

> 💡 자체 상품의 최대 장점: **수수료 없이 100% 내 수익!**\
> 애드센스는 구글이 일부 가져가지만,\
> 자체 상품은 전액이 내 것입니다.

## 확장 전략 — 건물을 더 크게!

수익화가 시작되면, **더 크게 키우는** 전략입니다.

### 전략 1: 사이트 하나 더 만들기

이미 한 번 만들어봤잖아요? 두 번째는 **더 빠릅니다.**

- 1호 건물: 테크/IT 블로그
- 2호 건물: 건강/다이어트 블로그
- 3호 건물: 여행/맛집 블로그

**리스크 분산**이에요. 한 분야가 안 되어도 다른 건물이 버텨줍니다.

### 전략 2: 콘텐츠 재활용

블로그 글 하나로 **5가지 콘텐츠**를 만들 수 있어요!

| 원본 | 변환 | 플랫폼 |
|------|------|--------|
| 블로그 글 | 숏폼 영상 | 유튜브 Shorts, 릴스 |
| 블로그 글 | 카드 뉴스 | 인스타그램 |
| 블로그 글 | 트윗/스레드 | X(트위터) |
| 블로그 글 | 뉴스레터 | 이메일 |
| 블로그 글 | 전자책 챕터 | 자체 상품 |

이것도 AI한테 시키면 됩니다:

```
이 블로그 글을 유튜브 쇼츠 대본으로 바꿔줘.
30초 분량, 후킹 → 핵심 3개 → CTA.
```

### 전략 3: 기존 글 업데이트

새 글만 쓰는 게 아닙니다!\
**이미 있는 글을 더 좋게 만드는 것**도 효과적이에요.

구글 서치 콘솔에서 각 글의 순위를 확인하세요:

| 현재 순위 | 할 일 |
|----------|-------|
| 1~3위 | 건드리지 마세요. 잘 되고 있어요 |
| **4~10위** | ⭐ **내용 보강하면 1~3위 가능!** 가장 효율적 |
| 11~20위 | FAQ 추가, 내용 확장하면 1페이지 진입 가능 |
| 20위+ | 키워드 재검토 필요 |

> 💡 **4~10위 글이 가장 효율적**입니다.\
> 조금만 보강하면 1페이지 상단으로 올라가요!

### 전략 4: 이메일 구독자 모으기

블로그 방문자를 **이메일 구독자**로 바꿉니다.

왜? 구글 SEO에만 의존하면 위험해요.\
이메일 구독자가 있으면 **구글 알고리즘과 상관없이** 트래픽을 만들 수 있습니다.

```
블로그 글 하단에 이메일 구독 폼을 추가해줘.
"매주 유용한 테크 팁을 보내드립니다" 문구와 함께.
```

## 현실적인 수익 타임라인

솔직하게 말씀드리겠습니다:

| 기간 | 글 수 | 예상 월 수익 | 현실 |
|------|-------|-----------|------|
| 1~3개월 | 10~20개 | 0~3만 원 | 거의 안 나옴. **여기서 90%가 포기** |
| 3~6개월 | 30~50개 | 5~20만 원 | 조금씩 보이기 시작 |
| 6~12개월 | 50~100개 | 20~100만 원 | "어, 진짜 되네?" 하는 순간 |
| 1년+ | 100개+ | 50만~수백만 원 | 시스템 완성, 복리 효과 |

> ⚠️ **처음 1~3개월은 인내의 시간입니다.**\
> 이때 포기하지 마세요. 씨앗을 심은 거예요.\
> **6개월만 꾸준히 하면 복리가 터지기 시작합니다.**

## 흔한 실패 패턴 — 이것만 피하세요!

| 실패 패턴 | 해결법 |
|----------|--------|
| "나중에 해야지" — 글을 안 씀 | **이번 주에 1개만** 쓰세요. 1개가 0개보다 무한히 낫습니다 |
| "완벽한 글을 써야지" — 한 글에 일주일 | **80점짜리를 빨리 많이** 올리세요. 나중에 업데이트하면 됩니다 |
| "2개월 했는데 돈이 안 나와요" | 2개월은 씨앗 단계. **6개월은 참으세요** |
| "디자인부터 완벽하게" — 글은 0개 | **글이 먼저!** 글이 돈을 법니다. 디자인은 아닙니다 |
| 트렌드 키워드만 쫓기 | **에버그린 70% + 트렌드 30%** 비율로 |
