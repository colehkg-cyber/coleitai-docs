# 6강. 그로스 해킹

> **3개월 만에 70만 사용자.** AARRR, 바이럴 루프, A/B 테스트 — 돈 없이 성장하는 법.

---

## AARRR 해적 지표

그로스의 뼈대 — **AARRR 퍼널(해적 지표)**.

| 단계 | 이름 | 질문 | 콘텐츠 적용 |
|------|------|------|------------|
| **A** | Acquisition (획득) | 유저가 어디서 오는가? | SEO 검색 유입 → 구글 → 내 글 |
| **A** | Activation (활성화) | "아하!" 순간이 있는가? | 글 읽고 "오 이거 좋다!" → 3분+ 체류 |
| **R** | Retention (유지) | 다시 돌아오는가? | 북마크, 구독, 재방문 |
| **R** | Revenue (수익) | 돈이 되는가? | 애드센스 클릭, 제휴 구매 |
| **R** | Referral (추천) | 남에게 알리는가? | "이 블로그 좋아" → SNS 공유 |

:::warning ⚠️ 핵심
퍼널 전체를 보라. **획득만 많고 유지가 안 되면 = 밑 빠진 독에 물 붓기.**
어디가 막히는지 데이터로 찾고, 거기를 집중 개선. 이게 그로스 해킹.
:::

## 바이럴 루프

유저가 유저를 데려오는 구조:

```
1. 유저가 콘텐츠 발견 (구글 검색)
   ↓
2. 가치를 느낌 ("이거 진짜 도움 된다!")
   ↓
3. 공유/추천 (카톡, SNS, 커뮤니티)
   ↓
4. 새로운 유저 유입 (공유 받은 사람이 클릭)
   ↓
5. 반복 → 루프!
```

### K-Factor (바이럴 계수)

```
K = 초대 수 × 전환율

K > 1 → 자동 성장 (유저가 알아서 늘어남)
K < 1 → 추가 마케팅 필요
```

**콘텐츠 바이럴 전략:**
공유하고 싶은 글 = 바이럴.
→ 실용적 + 구체적 + 공감 = "이거 나도 보내야겠다"

## A/B 테스트

**두 버전을 비교해서 더 나은 걸 선택.** 감 말고 데이터로 결정.

### 콘텐츠 A/B 테스트 예시

| 테스트 항목 | A (기존) | B (변경) | 결과 |
|------------|----------|----------|------|
| 글 제목 | "맥북 배터리 팁 7가지" | "맥북 배터리 2배 늘리는 법" | B가 CTR 23%↑ |
| CTA 위치 | 글 하단 | 글 중간 | 중간이 전환율 2배 |
| 썸네일 | 텍스트만 | 이미지 포함 | 이미지가 클릭률 35%↑ |
| 글 길이 | 2,000자 | 3,500자 | 3,500자가 체류시간 2배 |

**A/B 테스트 규칙:**
1. 한 번에 **하나만** 바꿔라
2. 최소 **2주**, **100클릭+** 데이터 수집
3. 감 → 데이터. 항상.

## PMF & 리텐션

### 리텐션 없는 성장 = 죽음 💀

```
유저 1만명 오는데 아무도 안 돌아옴
→ 매달 1만명을 새로 데려와야 함
→ 비용 폭발 → 지속 불가능
= "밑 빠진 독에 물 붓기"
```

### 리텐션 있는 성장 = 복리 💚

```
방문자가 북마크하고 재방문
→ 기존 유저 + 새 유저 = 복리 성장
→ 비용 거의 0 → 지속 가능
= "물이 차는 독"
```

### 콘텐츠 리텐션 전략

| 전략 | 방법 | 효과 |
|------|------|------|
| **시리즈 글** | "맥북 팁 1편, 2편, 3편" | 다음 편 기대감 → 재방문 |
| **내부 링크** | 글 안에 관련 글 링크 | 페이지뷰 2~3배 |
| **뉴스레터** | 새 글 알림 발송 | 구글 의존 탈피 |
| **에버그린** | 유행 안 타는 주제 | 꾸준한 재방문 |

## 데이터 드리븐 — 5가지 핵심 지표

매주 금요일 30분, 이것만 확인하세요:

| 지표 | 의미 | 도구 | 기준 |
|------|------|------|------|
| **트래픽** | 방문자 수 | GA, Search Console | 월 단위 추이 확인 |
| **체류시간** | 평균 세션 시간 | GA | 2분 이하 = 문제 |
| **CTR** | 클릭률 | 서치 콘솔 | 3% 이하 = 제목 수정 |
| **RPM** | 1,000 PV당 수익 | 애드센스 | $5~$30 (분야별) |
| **이탈률** | 한 페이지만 보고 떠남 | GA | 70%+ = 내부 링크 부족 |

## 그로스 플레이북 — 6가지 실전 전략

| # | 전략 | 방법 | 효과 |
|---|------|------|------|
| 1 | **제목 A/B** | 같은 글, 제목만 바꿔서 CTR 비교 | CTR 20~40%↑ |
| 2 | **80/20 법칙** | 상위 20% 글 집중 업데이트 | 적은 노력, 큰 효과 |
| 3 | **Internal Linking** | 모든 글에 관련 글 2~3개 링크 | 페이지뷰 2배 |
| 4 | **Content Gap** | 경쟁자는 있는데 나는 없는 키워드 | 빈틈 = 기회 |
| 5 | **시즌 선점** | "2025 맥북 추천" 미리 작성 | 시즌에 트래픽 폭발 |
| 6 | **클러스터** | 하나의 주제로 10개+ 글 | Topical Authority 구축 |
