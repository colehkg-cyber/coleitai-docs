# 8강. 크로스펑셔널 리더십

> **전부를 이해하면 전부를 엮을 수 있다.** T자형 인재 = 1인 빌더 = AI 시대의 최강 무기.

---

## T자형 인재

| 유형 | 설명 | 장단점 |
|------|------|--------|
| **I자형** | 한 분야만 깊음 | 전문가지만, 협업 어렵고 시야 좁음 |
| **T자형** | 넓게 이해 + 하나 깊게 | 실리콘밸리 표준. 팀의 허브 역할 |
| **π(파이)자형** | 넓게 이해 + 둘 깊게 | 미래의 인재상. 최강의 1인 빌더 |

```
I자형:     T자형:        π자형:
  │        ━━━━━━━       ━━━━━━━
  │           │            │   │
  │           │            │   │
  │           │            │   │
(한 분야)  (넓은 이해    (넓은 이해
           + 깊은 전문)  + 2개 전문)
```

:::tip 🎯 기초편을 다 들으면?
여러분은 이미 T자형의 **"ㅡ" 부분**을 갖게 됩니다.
디자인, 기획, 개발, 마케팅, 세일즈 — 전부 이해하는 사람.
**이제 본편에서 "ㅣ"를 세울 시간입니다.**
:::

---

## 1인 빌더 시대 — AI × 크로스펑셔널

과거에는 6명이 할 일을, AI + 크로스펑셔널 1명이 다 합니다.

| 역할 | AI 도구 | AI 기여도 |
|------|---------|----------|
| 🎨 디자인 | Midjourney, Figma AI | ~80% |
| 💻 개발 | Cursor, Claude Code | ~90% |
| ✍️ 콘텐츠 | ChatGPT, Claude | ~70% |
| 📊 분석 | GA, Search Console | 도구가 해줌 |
| 📢 마케팅 | SEO + SNS 자동화 | 시스템이 해줌 |
| 💰 세일즈 | 제휴 링크, 애드센스 | 시스템이 해줌 |

> **과거:** 6명 팀 필요 → **현재:** AI + 1인 = 6인 역할.
> 크로스펑셔널 이해가 기반이 되어야 AI를 제대로 활용할 수 있다.
