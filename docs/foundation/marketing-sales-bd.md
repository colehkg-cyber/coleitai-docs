# 7강. 마케팅 × 세일즈 × BD

> **만든 건 팔아야 의미가 있다.** CAC, LTV, 퍼포먼스 마케팅 — 수익의 진짜 원리.

---

## CAC & LTV — 사업의 건강 지표

### CAC (Customer Acquisition Cost)
**고객 1명 데려오는 비용.**

```
CAC = 마케팅 비용 ÷ 신규 고객 수

광고비 100만원 → 고객 100명 = CAC 1만원
콘텐츠 마케팅(SEO) = CAC ≈ 0원! ← 핵심 장점
```

### LTV (Lifetime Value)
**고객 1명이 평생 가져다주는 가치.**

```
LTV = 월 수익 × 평균 유지 기간

블로그 방문자: 재방문 × 광고 수익 + 제휴 구매
```

### Unit Economics

```
LTV > CAC = 건강한 사업
LTV / CAC ≥ 3 = 아주 건강!
```

| | 콘텐츠 사업 |
|---|---|
| 글 1개 작성 비용 | 1시간 (≈1만원) |
| 글 1개 연간 수익 | 5~50만원 |
| ROI | 500~5,000% |
| CAC | ≈ 0원 (SEO 유입) |
| LTV | 높음 (글이 영원히 일함) |
| 마진 | 거의 100% |

:::tip 💡 이것이 콘텐츠 사업이 강력한 이유
CAC ≈ 0, LTV = 높음, 마진 ≈ 100%. 다른 사업 모델에서는 불가능한 구조.
:::

## 마케팅 채널

| 채널 | CAC | 속도 | 지속성 | 콘텐츠 적합도 |
|------|-----|------|--------|-------------|
| **SEO** | 0원 | 느림 (3~6개월) | ⭐ 장기, 복리 | 핵심 채널 |
| **SNS** | 0~낮음 | 빠름 | 노력 비례 | 리사이클링용 |
| **퍼포먼스 광고** | 높음 | 즉시 | 돈 멈추면 0 | 테스트용만 |
| **이메일** | 매우 낮음 | 중간 | 구글 의존 탈피 | 장기 전략 |

**결론:** SEO(핵심) + SNS(보조) + 이메일(장기). 광고 의존은 위험.

---

## 세일즈 퍼널

방문자를 고객으로 바꾸는 구조:

```
인지 (100명 방문)
  ↓ 30%
관심 (30명 다른 글도 읽음)
  ↓ 33%
고려 (10명 구독)
  ↓ 30%
구매 (3명 제휴 구매)
  ↓ 33%
충성 (1명 공유 → 바이럴)
```

:::tip 🎯 핵심
각 단계의 전환율을 **10%만 올려도** 최종 매출은 **60%+** 증가.
퍼널 전체를 데이터로 관리하세요.
:::

## BD (사업개발)

혼자 하지 마라 — 파트너와 함께 키워라.

| BD 유형 | 방법 | 효과 |
|---------|------|------|
| **제휴 파트너십** | 쿠팡 파트너스, SaaS 제휴 | 수수료 = 패시브 수익 |
| **게스트 포스팅** | 다른 블로그에 글 기고 | 백링크 획득 = SEO 부스트 |
| **콜라보 마케팅** | 유사 분야 블로거와 협업 | Win-Win 성장 |
| **번들 딜** | 다른 크리에이터와 상품 묶음 | 판매량 × 2 |

### BD 콜드메일 공식

```
[Hook]    상대방 칭찬 1줄
[Value]   내가 줄 수 있는 가치
[Ask]     구체적 요청
[Easy Out] 부담 없는 마무리
```

**예시:**
> "○○님 블로그에서 '맥북 배터리' 글 잘 봤습니다. (Hook)
> 저도 맥북 관련 블로그를 운영하고 있는데, 서로 글에 링크를 교차하면 양쪽 다 SEO에 도움이 될 것 같습니다. (Value)
> 혹시 관심 있으시면 구체적으로 이야기 나눌 수 있을까요? (Ask)
> 바쁘시면 무시하셔도 전혀 괜찮습니다! (Easy Out)"

## 콘텐츠 마케팅 전략 — TOFU / MOFU / BOFU

| 단계 | 키워드 유형 | 목적 | 배분 비율 |
|------|-----------|------|----------|
| **TOFU** (Top of Funnel) | 넓은 키워드 ("맥북 추천") | 트래픽 확보 | 50% |
| **MOFU** (Middle of Funnel) | 구체적 키워드 ("맥북 에어 M3 후기") | 신뢰 구축 | 30% |
| **BOFU** (Bottom of Funnel) | 구매 키워드 ("맥북 에어 최저가") | 전환/수익 | 20% |

:::tip 💡 콘텐츠 배분 공식
TOFU 50% + MOFU 30% + BOFU 20% = 건강한 콘텐츠 포트폴리오.
:::
