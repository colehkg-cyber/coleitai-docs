| 레이어 | 기술 | 역할 | 비용 |

| **보안** | Vercel (자동) | HTTPS 인증서 | 무료 |
| **합계** | | | **월 $20** |

## 배포 파이프라인

```
나: 글 작성 or 코드 수정
    │
    ▼
git add . && git commit && git push
    │
    ▼
GitHub: 코드 변경 감지
    │
    ▼
Vercel: 자동 빌드 시작 (약 30초~1분)
    │
    ├── Next.js 빌드 (SSG)
    ├── 이미지 최적화
    ├── sitemap.xml 업데이트
    └── 전 세계 CDN에 배포
    │
    ▼
내 사이트: 업데이트 완료! ✅
```

> 💡 **기존 방식 (AWS EC2)** 대비:\
> 월 5~10만원 절감 + 서버 터질 걱정 없는 Serverless 구조.

## 템플릿 구조

```
my-blog/
├── public/              # 정적 파일 (이미지, robots.txt 등)
│   ├── images/
│   ├── robots.txt
│   └── sitemap.xml
├── posts/               # 블로그 글 (마크다운)
│   ├── 첫번째-글.mdx
│   └── 두번째-글.mdx
├── components/          # 재사용 UI 컴포넌트
├── styles/              # CSS 스타일
├── lib/                 # 유틸리티 함수
├── pages/               # 페이지 라우팅
│   ├── index.tsx        # 메인 페이지
│   ├── posts/[slug].tsx # 글 상세 페이지
│   └── about.tsx        # 소개 페이지
├── next.config.js       # Next.js 설정
├── package.json         # 의존성 관리
└── README.md
```

## SEO 자동화 구조

```
글 작성 (.mdx)
    │
    ├── 메타 태그 자동 생성 (title, description, OG)
    ├── Schema.org 마크업 자동 삽입
    ├── sitemap.xml 자동 업데이트
    ├── 이미지 WebP 자동 변환
    ├── H1 중복 자동 방지
    └── 반응형 레이아웃 자동 적용
    │
    ▼
Lighthouse 400점 구조 유지 ✅
```
