# 콘텐츠 SEO 이해하기

> 기초 공사(테크니컬 SEO) 끝. 이제 입주민(콘텐츠)을 들여야 월세가 나옵니다.

---

## 콘텐츠 SEO란?

**사람들이 검색하는 키워드에 맞는 글을 써서, 구글 검색 상위에 노출시키는 것입니다.**

쉽게 말하면:

| 비유 | 테크니컬 SEO | 콘텐츠 SEO |
|------|------------|-----------|
| 건물 | 기초 공사 완료 ✅ | 이제 세입자(글)를 들이자 |
| 역할 | 구글 봇이 좋아하는 **구조** | 사람이 좋아하는 **내용** |
| 빈도 | 한 번 세팅하면 끝 | 꾸준히 글을 쓴다 |
| 누가? | 템플릿이 해결 | **여러분이 하는 것** |

> 💡 **건물(사이트)은 완성했습니다.** 이제 세입자(글)가 필요합니다.\
> 세입자가 없는 건물에서는 월세가 안 나오잖아요?\
> 글이 없는 사이트에서는 트래픽(방문자)이 안 옵니다.

---

## 황금 키워드 공식

이것이 구글 건물주 프로젝트의 **가장 중요한 전략**입니다.

```
┌──────────┐     ┌──────────┐     ┌──────────────┐
│ 검색량    │  ×  │ 경쟁도    │  =  │  황금 키워드   │
│   HIGH   │     │   LOW    │     │  🏆 대박!     │
└──────────┘     └──────────┘     └──────────────┘
```

무슨 뜻이냐면:

- **검색량이 높다** = 사람들이 이 키워드로 많이 검색한다
- **경쟁도가 낮다** = 이 키워드에 대해 좋은 글이 별로 없다
- **두 조건이 동시에 충족** = 여기에 글을 쓰면 1페이지 장악 가능!

이런 키워드를 **"빈 땅"**이라고 부릅니다.\
사람들은 찾아오는데, 건물(좋은 글)이 없는 땅.\
여기에 400점짜리 사이트로 좋은 글을 올리면? **독점!**

### 실전 예시

```
키워드: "맥북 에어 M3 발열 해결법"
─────────────────────────────────
검색량:    월 2,400회 → 사람들이 꽤 많이 검색함 ✅
경쟁 상태:  상위 글들이 3년 전 글, 내용 부실 ✅
기존 글:   오래되고 정보가 틀림 ✅
─────────────────────────────────
→ 황금 키워드 확정! 🏆
→ 최신 정보로 글을 쓰면 1페이지 진입 가능
```

---

## 검색 의도 — 사람들은 왜 검색하는가?

같은 "맥북"이라는 단어로 검색해도, 사람마다 원하는 게 다릅니다.\
이걸 **검색 의도**라고 합니다.

| 유형 | 사람의 마음 | 검색 예시 | 우리가 쓸 글 스타일 |
|------|-----------|----------|-----------------|
| **정보형** | "이게 뭔지 알고 싶다" | "맥북 에어 발열 원인" | 설명 가이드 |
| **비교형** | "뭐가 더 나은지 알고 싶다" | "맥북 에어 vs 프로" | 상세 비교표 |
| **해결형** | "문제를 풀고 싶다" | "맥북 발열 해결법" | 해결 방법 글 |
| **구매형** | "사고 싶다" | "맥북 에어 최저가" | 리뷰 + 구매 링크 |

### 우리가 노려야 할 유형은?

**정보형 + 해결형**이 가장 좋습니다.

왜?
- 비교형, 구매형은 새 제품이 나오면 글이 쓸모없어져요
- 정보형/해결형은 **오래 갑니다** (글 1개 = 월세 1건이 제대로 작동!)

> 💡 물론 비교형, 구매형도 쓰면 좋습니다.\
> 특히 제휴 마케팅(제품 추천해서 수수료 받기)에는 이런 글이 효과적이에요.\
> 하지만 **메인은 정보형/해결형**으로 가는 게 안정적입니다.

## AI 글쓰기의 핵심

### Claude에게 글을 시키는 흐름

```
1. 황금 키워드를 찾는다
       ↓
2. Claude에게 "이 키워드로 블로그 글 써줘" 한다
       ↓
3. Claude가 초안을 써줌
       ↓
4. 내가 읽어보고, 개인 경험을 추가한다 ← 이게 핵심!
       ↓
5. 블로그에 올린다 → 자동 배포!
```

### "AI가 쓴 글을 올려도 되나요?"

구글의 공식 입장:

> "우리는 콘텐츠가 **어떻게** 만들어졌는지가 아니라,\
> 사용자에게 **유용한지**를 평가합니다." — Google, 2023

**결론:** AI로 쓰되, 사람이 검수하고 **개인 경험을 추가하면 OK!**

중요한 건 글의 퀄리티입니다.\
AI가 쓴 글 그대로 올리면 밋밋해서 구글이 안 좋아할 수 있어요.\
하지만 거기에 **"제가 직접 써봤는데..."** 같은 경험을 넣으면?\
유일무이한, 가치 있는 글이 됩니다.

## 글 하나의 수명 — 왜 구글이 네이버보다 나은가

이게 가장 중요합니다. 잘 보세요!

### 네이버 블로그 글
```
트래픽 ↑
      │  ╱╲
      │ ╱  ╲──── 2~4주 후 사라짐...
      │╱    ╲
      └──────────── 시간 →
```
글 올리면 2~4주 반짝 → 그다음 트래픽 0 → **또 써야 함**\
글 안 쓰면 수입 0. 노동집약적이에요.

### 구글 SEO 글
```
트래픽 ↑
      │            ╱────── 6개월~수년 유지!
      │          ╱
      │        ╱
      │      ╱
      │    ╱
      └──────────── 시간 →
```
처음 1~2달은 천천히 올라감 → 한 번 올라가면 **계속 유지!**\
6개월, 1년, 3년... 꾸준히 트래픽이 들어옴

> 🏠 **글 1개 = 월세 1건입니다.**\
> 글이 쌓일수록 월세가 늘어납니다.\
> 이게 "구글 건물주"의 본질이에요.\
> 글을 쌓아가면 **복리**처럼 트래픽이 불어납니다.

---

## 글이 쌓이면 어떻게 되나요?

| 글 수 | 예상 월 트래픽 | 비유 |
|-------|-------------|------|
| 10개 | 500~1,000명 | 원룸 건물 |
| 30개 | 3,000~5,000명 | 소형 빌딩 |
| 50개 | 10,000~20,000명 | 중형 빌딩 |
| 100개+ | 50,000명+ | 대형 빌딩 |

글 1개 1개가 다 세입자입니다.\
**세입자가 많을수록 월세가 늘어납니다.** 꾸준히 쓰세요!
