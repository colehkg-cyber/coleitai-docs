# 콘텐츠 SEO 이해하기

> 기초 공사(테크니컬 SEO) 끝. 이제 입주민(콘텐츠)을 들여야 월세가 나옵니다.

---

## 콘텐츠 SEO란?

**사람들이 검색하는 키워드에 맞는 양질의 글을 써서, 구글 검색 상위에 노출시키는 것.**

| 비유 | 테크니컬 SEO | 콘텐츠 SEO |
|------|------------|-----------|
| 건물 | 기초 공사 완료 ✅ | 이제 세입자(글)를 들이자 |
| 역할 | 구글 봇이 좋아하는 구조 | 사람이 좋아하는 내용 |
| 빈도 | 한 번 세팅하면 끝 | 지속적으로 글 발행 |

---

## 황금 키워드 공식

이것이 구글 건물주 프로젝트의 **핵심 전략**입니다.

```
┌──────────┐     ┌──────────┐     ┌──────────────┐
│ 검색량    │  ×  │ 경쟁도    │  =  │  황금 키워드   │
│   HIGH   │     │   LOW    │     │ 트래픽 독점!  │
└──────────┘     └──────────┘     └──────────────┘
```

> 사람들이 검색은 하는데, 읽을 만한 문서는 없는 **'빈 땅'**을 찾아냅니다.

### 3단계 프로세스

| 단계 | 이름 | 설명 |
|------|------|------|
| **1** | 데이터 마이닝 | 구글 트렌드, 키워드 플래너로 월간 검색량 1,000 이상 키워드 추출 |
| **2** | 문서 공백 발견 | 상위 노출된 글들이 '오래되었거나' '내용이 부실한' 키워드 필터링 |
| **3** | AI 콘텐츠 투입 | 400점짜리 구조에 고품질 정보를 채워 1페이지 장악 |

### 실전 예시

```
키워드: "맥북 에어 M3 발열 해결법"
─────────────────────────────────
검색량:    월 2,400회 ✅
경쟁도:    검색 결과 부실 ✅
기존 글:   3년 전 작성, 구버전 정보 ✅
─────────────────────────────────
→ 황금 키워드 확정!
→ 최신 정보로 글 작성 → 1페이지 진입
```

## 검색 의도 (Search Intent)

같은 키워드라도 사람마다 원하는 게 다릅니다.\
검색 의도를 맞춰야 체류 시간이 늘고, 구글 순위가 올라갑니다.

| 의도 유형 | 예시 | 글 전략 | 수익성 |
|----------|------|---------|--------|
| **정보형** | "SEO란 무엇인가" | 개념 설명 가이드 | ★★☆ |
| **비교형** | "맥북 vs 갤럭시북" | 상세 비교표 | ★★★ |
| **구매형** | "에어팟 프로 최저가" | 리뷰 + 구매 링크 | ★★★ |
| **리스트형** | "AI 도구 추천 TOP 10" | 순위 리스트 | ★★★ |

> 💡 **비교형, 리스트형**이 수익성이 가장 높습니다.\
> 제휴 마케팅과 직접 연결되기 때문입니다.

## AI 글쓰기의 핵심

### Claude Code로 글 쓰는 흐름

```
"야, 이 키워드로 블로그 글 써줘"
        ↓
   AI가 초안 작성
        ↓
   내가 검수 + 경험 추가
        ↓
   발행 → 구글 상위 노출
```

### 구글의 AI 콘텐츠 공식 입장

> "우리는 콘텐츠가 **어떻게** 만들어졌는지가 아니라,\
> 사용자에게 **유용한지**를 평가합니다." — Google, 2023

**결론:** AI로 쓰되, 반드시 사람이 검수하고 개인 경험을 추가하면 OK.

## 글 하나의 수명

### 네이버 블로그 글

```
수익 ↑
     │  ╱╲
     │ ╱  ╲──── (2~4주 후 사라짐)
     └──────── 시간 →
```

### 구글 SEO 글

```
수익 ↑
     │         ╱────── (6개월~ 수년 유지)
     │       ╱
     │     ╱
     │   ╱
     └──────── 시간 →
```

> **글 1개 = 월세 1건.**\
> 글이 쌓일수록 월세가 늘어납니다. 이게 "구글 건물주"의 본질입니다.
