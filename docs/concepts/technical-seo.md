# 테크니컬 SEO 이해하기

> 건물의 인테리어(콘텐츠)보다 먼저, 기초 공사(테크니컬 SEO)가 되어야 합니다.

---

## 테크니컬 SEO란?

**검색엔진(구글 봇)이 내 사이트를 잘 읽고, 좋아할 수 있도록 기술적으로 최적화하는 것입니다.**

어렵게 느껴지시죠? 쉽게 비유해볼게요:

| 비유 | 콘텐츠 SEO | 테크니컬 SEO |
|------|----------|-------------|
| 건물 | 인테리어, 가구 (예쁜 벽지, 좋은 가구) | 기초 공사, 철근 구조 (튼튼한 뼈대) |
| 음식점 | 맛있는 요리 | 위생 등급, 주방 시설 |
| 자동차 | 외관 디자인 | 엔진 성능, 안전 등급 |

> 💡 **쉽게 말하면:**\
> 아무리 맛있는 식당이라도 **위생 등급이 D**면 손님이 안 오잖아요?\
> 구글도 마찬가지입니다.\
> 글이 아무리 좋아도 **기술 점수가 낮으면 상위 노출을 안 시켜줍니다.**

### 그래서 이걸 내가 해야 하나요?

**아닙니다!** 이게 이 강의의 핵심이에요.

보통 SEO 강의에서는 "콘텐츠 SEO"만 가르칩니다.\
테크니컬 SEO는? "그건 개발자한테 맡기세요~" 이러죠.\
왜? **어려우니까.**

하지만 이 강의에서는 **템플릿이 다 해결합니다.**\
여러분이 코드를 건드릴 일이 없어요.\
설치만 하면 테크니컬 SEO가 **자동으로** 적용됩니다.

---

## 구글 봇이 뭔가요?

"구글 봇"이라는 프로그램이 있습니다.\
이게 **전 세계 모든 웹사이트를 24시간 돌아다니면서 점수를 매깁니다.**

여러분이 구글에서 뭔가를 검색하면 결과가 나오잖아요?\
그 순서를 구글 봇이 정합니다. 점수가 높은 사이트가 위에 나와요.

### 구글 봇이 보는 4가지

```
구글 봇이 내 사이트에 방문
    ↓
"이 사이트 빠른가?"          → Performance (성능)
"모든 사람이 쓸 수 있는가?"   → Accessibility (접근성)
"웹 표준을 지키는가?"         → Best Practices (모범 사례)
"검색엔진이 잘 읽는가?"       → SEO
    ↓
4가지 점수 합산 = Lighthouse Score (최대 400점)
    ↓
점수 높을수록 → 검색 결과 상위 노출!
```

> 💡 각각 100점 만점이에요. 합치면 **400점 만점**.\
> 이걸 측정하는 도구가 구글이 직접 만든 **Lighthouse**(라이트하우스)입니다.

---

## 핵심 개념 4가지 (쉽게 설명)

### 1. Core Web Vitals — "사이트가 빠른가?"

구글이 **가장 중요하게 보는** 3가지 성능 지표입니다.\
어려워 보이지만, 쉽게 설명하면 이렇습니다:

| 지표 | 쉬운 뜻 | 비유 | 합격 기준 |
|------|---------|------|----------|
| **LCP** | 화면이 뜨는 데 걸리는 시간 | "문 열었는데 불이 언제 켜지냐" | 2.5초 이하 |
| **FID** | 버튼을 눌렀을 때 반응 속도 | "버튼 눌렀는데 언제 반응하냐" | 0.1초 이하 |
| **CLS** | 화면이 갑자기 움직이는 정도 | "글 읽는데 갑자기 광고가 끼어들어서 화면이 밀리는 거" | 0.1 이하 |

:::warning
**CLS가 가장 흔한 감점 원인입니다.**

글을 읽고 있는데 갑자기 광고가 뜨면서 화면이 밀리는 거 겪어보셨죠?\
그게 CLS입니다. 이미지에 크기 설정을 안 하면 발생합니다.

→ 걱정 마세요! **우리 템플릿은 이걸 자동으로 잡아줍니다.**
:::

### 2. 시맨틱 HTML — "사이트 구조가 정리되어 있는가?"

"시맨틱 HTML"이란, 구글 봇한테\
**"여기가 제목이야, 여기가 본문이야, 여기가 하단이야"**\
알려주는 구조입니다.

쉽게 비유하면:

- **나쁜 구조** = 서류가 바닥에 아무렇게나 흩어져 있는 사무실\
  → 구글 봇: "이게 뭔지 모르겠는데..."
- **좋은 구조** = 서류가 폴더별로 깔끔하게 정리된 사무실\
  → 구글 봇: "오, 깔끔하다! 이 사이트 좋은데?"

> 💡 여러분이 이 구조를 직접 만들 필요 없습니다.\
> **템플릿에 다 들어있습니다.** 자동이에요.

### 3. 메타 태그 — "이 사이트가 뭔지 알려주는 명함"

구글에서 뭔가를 검색하면, 각 결과에 **제목**과 **설명**이 보이잖아요?\
그게 메타 태그입니다.

이게 잘 되어 있으면 **클릭률이 3%에서 15%까지** 올라갑니다.\
같은 순위라도 설명이 매력적이면 더 많이 클릭하는 거죠.

> 💡 **우리 템플릿은 메타 태그를 자동으로 만들어줍니다.**\
> 글 제목을 쓰면 자동으로 검색 결과용 제목과 설명이 생성됩니다.

### 4. 사이트맵 & robots.txt — "구글에 지도 제출"

| 파일 | 쉽게 말하면 | 비유 |
|------|-----------|------|
| **사이트맵** (sitemap.xml) | 내 사이트에 어떤 글들이 있는지 목록 | 건물 도면을 시청에 제출하는 것 |
| **robots.txt** | 구글 봇한테 "우리 사이트 다 봐도 돼" 알려주는 파일 | 건물 출입 허용 안내문 |

> 💡 이것도 **우리 템플릿에 다 포함**되어 있습니다.\
> 글을 추가할 때마다 사이트맵이 자동으로 업데이트됩니다.

## 테크니컬 SEO vs 콘텐츠 SEO — 정리

| | 테크니컬 SEO | 콘텐츠 SEO |
|---|---|---|
| **비유** | 건물의 기초/구조 | 건물의 인테리어/입주민 |
| **작업** | 사이트 속도, 구조, 접근성 | 키워드 선정, 글쓰기 |
| **빈도** | 처음 한 번 세팅하면 끝 ✅ | 지속적으로 글 작성 |
| **누가 하나?** | **템플릿이 자동으로!** ✅ | 여러분이 하는 것 |

> 🎯 **이 강의의 핵심:**\
> 테크니컬 SEO는 **템플릿**이 다 해결합니다.\
> 여러분은 **콘텐츠(글)**에만 집중하면 됩니다.
