# 왜 구글 건물주인가

> 남의 땅에서 전세살이 그만하고, 내 건물을 짓자.

---

## 먼저, "구글 건물주"가 뭔가요?

아주 간단합니다:

1. **인터넷에 나만의 사이트(건물)를 만든다**
2. **글을 쓴다** (이게 세입자)
3. **구글 검색을 통해 사람들이 찾아온다** (이게 트래픽)
4. **방문자에게 광고를 보여주고 수익을 받는다** (이게 월세)

여기서 핵심은 **"내 것"**이라는 겁니다.\
네이버 블로그는 네이버 거예요. 티스토리는 카카오 거고요.\
하지만 우리가 만드는 사이트는 **100% 내 것**입니다.

---

## 핵심 비유: 부동산

블로그를 부동산에 비유하면 모든 게 명확해집니다.

| | 플랫폼 블로그 | 구글 건물주 |
|---|---|---|
| **비유** | 🏢 남의 건물에 전세 | 🏠 내 땅에 내 건물 |
| **대표** | 네이버 블로그, 티스토리, 브런치 | 나만의 독립 사이트 |
| **소유권** | 플랫폼 것 (내 것 아님!) | 100% 내 자산 |
| **노출** | 플랫폼 맘대로 바뀜 | 구글에서 영구적으로 상위 노출 |
| **리스크** | 저품질 한 방이면 노출 0 | 내가 삭제 안 하면 영구적 |
| **수익** | 제한적 (광고 단가 낮음) | 무제한 확장 가능 |
| **서버비** | 무료 (대신 자유도 없음) | **역시 무료!** |

> 💡 **"네이버 블로그도 무료인데 뭐가 다른가요?"**\
> 네이버 블로그는 무료 전세입니다. 건물주(네이버)가 "나가" 하면 나가야 해요.\
> 구글 건물주는 **내 땅에 내 건물**입니다. 아무도 못 내보냅니다.

## 실제 데이터

```
📊 구글 건물주 프로젝트 실적
─────────────────────────────
월간 순방문자 (MAU)    200,000+
지난 1년간 신규 글       0건
월 평균 서버비           ₩0
순수익률                99%
─────────────────────────────
```

> **글을 안 써도 매달 20만 명이 찾아옵니다.**\
> 지난 1년간 글 한 개도 안 썼는데 트래픽이 유지됩니다.\
> 왜? 예전에 써놓은 글들이 **구글에서 계속 상위에 있으니까요.**

---

## 플랫폼 전세살이의 위험성

"에이, 네이버 블로그 잘 되는 사람도 많잖아요."

맞습니다. **지금은요.**\
하지만 이런 일들이 실제로 있었습니다:

### 😰 실제로 일어난 일들

**네이버 블로그**
- 저품질 알고리즘을 갑자기 바꿨습니다
- **3년 동안 매일 글 쓴 사람이 하루 만에 노출 0이 됐습니다**
- 구제 방법? 없습니다. 네이버가 안 된다고 하면 안 되는 거예요
- 내 데이터의 소유권도 없습니다 — 네이버 약관상 네이버 것입니다

**티스토리**
- 카카오가 에디터를 강제로 바꿨습니다 → 기존 글이 다 깨졌습니다
- 광고 수익 구조도 맘대로 바꿨습니다
- 만약 카카오가 서비스를 접으면? → **모든 것이 사라집니다**

**브런치**
- 이건 아예 이사를 못 갑니다. 콘텐츠 이전 기능이 없어요
- 수익 모델도 극히 제한적입니다

### 공통점이 뭔가요?

> **건물주(플랫폼)가 마음 바꾸면, 세입자(여러분)는 속수무책입니다.**\
> 그래서 내 건물을 지어야 합니다.

---

## 내 건물의 장점 4가지

### 1. ✅ 완전한 소유권
코드도 내 것, 데이터도 내 것, 트래픽도 내 것.\
누구도 내 사이트를 내릴 수 없습니다.\
네이버가 망해도, 카카오가 망해도, **내 사이트는 살아있습니다.**

### 2. ✅ 영구적인 상위 노출
구글 검색에 최적화해놓으면, 한 번 올라간 글은 **계속 유지**됩니다.

네이버 블로그는 글을 올리면 2~4주 반짝하고 사라집니다.\
구글은? 제 글 중에 **3년 전에 쓴 게 아직도 1페이지**에 있습니다.

### 3. ✅ 비용 0원
- **Github** (코드 저장) → 무료
- **Vercel** (사이트를 인터넷에 올려줌) → 무료
- **HTTPS 보안 인증서** → 무료

유일한 비용은 **Claude Pro 월 $20** (약 2만 7천 원).\
그게 전부입니다.

### 4. ✅ 무제한 확장성
- 구글 애드센스 (광고 수익)
- 제휴 마케팅 (제품 추천 수수료)
- 디지털 상품 판매 (전자책, 강의 등)
- 수익 파이프라인을 **얼마든지** 추가할 수 있습니다

## 글의 수명 — 이게 가장 중요합니다

### 네이버 블로그 글의 수명
```
트래픽 ↑
      │  ╱╲
      │ ╱  ╲──── 2~4주 후 사라짐
      │╱    ╲
      └──────────── 시간 →
```
글을 올리면 2~4주 반짝하고, 트래픽 0. 또 써야 합니다.\
**노동집약적**이에요. 글 안 쓰면 수입 0.

### 구글 SEO 글의 수명
```
트래픽 ↑
      │            ╱────── 6개월~ 수년 유지!
      │          ╱
      │        ╱
      │      ╱
      │    ╱
      └──────────── 시간 →
```
처음 1~2달은 천천히 올라갑니다. 하지만 한 번 올라가면?\
6개월, 1년, 3년... **계속** 트래픽이 들어옵니다.

> **글 1개 = 월세 1건입니다.**\
> 글이 쌓일수록 월세가 늘어납니다. 이게 "건물주"의 본질이에요.
